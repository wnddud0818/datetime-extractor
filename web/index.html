정적 룰 전용 페이지

datetime-extractor 플레이그라운드

Vercel 정적 배포용 페이지입니다. 백엔드와 API 없이 브라우저에서 바로 룰 엔진을 실행합니다.

브라우저 런타임 LLM 비활성화
이 페이지에서는 LLM 기능을 의도적으로 제외했습니다.

날짜 추출은 결정론적인 룰 엔진만 사용합니다. 원래 Ollama 폴백이 필요했던 기능은 비활성화 상태로 표시됩니다.

공휴일과 영업일 계산은 번들에 포함된 2024년부터 2030년까지의 공휴일 데이터를 사용합니다.

출력 모드 ?

원하는 결과 형태를 하나 이상 선택하세요.

지원 옵션 ?

브라우저 정적 페이지에서도 그대로 동작하는 옵션입니다.

이 페이지에서 비활성화된 기능 ?

정적 환경의 한계를 명확히 보여주기 위해 비활성화 상태로 표시합니다.

예제 ?

비활성 예제는 원래 LLM 의존도가 있던 케이스입니다.

브라우저 전용 런타임입니다. 이 페이지에서는 rule 또는 cache 경로만 나타나는 것이 정상입니다.

문장을 입력하고 추출을 실행하면 룰 엔진의 해석 결과를 여기서 확인할 수 있습니다.
